## 📐 SPESIFIKASI TEKNIS UMUM

**Format Output:**
- PNG dengan background (BUKAN transparent!)
- Resolusi: 960×544 pixels (landscape, full scene)
- Style: Ace Attorney / Phoenix Wright visual novel
- Anime/manga art style dengan outline tegas hitam
- Warna cel-shaded (flat colors dengan shading minimal)

**Sprite Sheet Layout:**
- Grid horizontal: N kolom × 1 baris (sesuai jumlah ekspresi)
- Setiap scene: 960×544 pixels (full background + character)
- Spacing antar scene: 32px padding
- Total canvas width akan adjust (contoh: 4 ekspresi = 4032×544px)

**Komposisi Per Scene:**
- Character + Background dalam 1 frame
- Karakter dari pinggang ke atas (waist-up)
- Background sesuai role (lihat detail per karakter)
- Tangan/gesture visible untuk ekspresi
- Konsisten: posisi, ukuran, dan proporsi SAMA di semua sprite

## 🎯 TIPS PENGGUNAAN GEMINI AI

1. **Generate 1 sprite sheet per karakter** - Semua ekspresi dalam 1 gambar
2. **Copy-paste prompt lengkap** - Jangan edit, langsung paste
3. **Perhatikan arah hadap:**
   - Hakim: Forward/Center
   - Jaksa: LEFT ←
   - Terdakwa: RIGHT →
   - Saksi: LEFT ←
4. **Jika hasil tidak sesuai**, tambahkan: "make it more like Ace Attorney game visual novel scene, with consistent character proportions and background across all scenes"
5. **Jika arah hadap salah**, tambahkan: "character must face [LEFT/RIGHT], 3/4 view to the [left/right]"
6. **Jika background tidak konsisten**, tambahkan: "same exact background in all scenes, only character expression changes"
7. **Save dengan naming convention**: `[nama]_spritesheet.png`
   - Contoh: `raka_spritesheet.png`, `hakim_spritesheet.png`
8. **Setelah download**, crop individual scenes menggunakan tool (lihat bagian berikutnya)

## � CARA CROP SPRITE SHEET

Setelah mendapat sprite sheet dari Gemini AI, crop menjadi individual sprites:

### Opsi 1: Online Tool (Mudah)
1. Buka: https://www.imgonline.com.ua/eng/cut-photo-into-pieces.php
2. Upload sprite sheet
3. Set: Columns = jumlah ekspresi, Rows = 1
4. Download hasil crop

### Opsi 2: Photoshop/GIMP (Manual)
1. Buka sprite sheet
2. Gunakan Slice Tool / Guides
3. Set grid: 256px width, 32px gutter
4. Export slices as individual PNG

### Opsi 3: ImageMagick (Command Line)
```bash
# Untuk 4 sprite (contoh: Hakim)
magick hakim_spritesheet.png -crop 256x384+0+0 hakim_netral.png
magick hakim_spritesheet.png -crop 256x384+288+0 hakim_marah.png
magick hakim_spritesheet.png -crop 256x384+576+0 hakim_serius.png
magick hakim_spritesheet.png -crop 256x384+864+0 hakim_terkejut.png
```
